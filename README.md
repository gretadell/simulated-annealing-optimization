# Simulated Annealing Optimization

This repository contains a programming project focused on simulated annealing for grid-based optimization.

The project implements an optimization problem on an n x n matrix and applies simulated annealing to search for the minimum of the function.

The analysis studies how temperature, beta values, acceptance probabilities, and grid size affect the performance of the algorithm.

The notebook includes code implementation, visualizations of the optimization landscape, and performance tests across different problem dimensions.

The results show how simulated annealing balances exploration and exploitation depending on the temperature schedule.

Particular attention is given to convergence speed, convergence success, and the risk of getting trapped in local minima.
